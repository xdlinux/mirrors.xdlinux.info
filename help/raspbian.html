使用说明

编辑 /etc/apt/sources.list 文件。删除原文件所有内容，用以下内容取代：

    deb http://mirrors.xdlinux.info/raspbian/raspbian/ wheezy main non-free contrib
    deb-src http://mirrors.xdlinux.info/raspbian/raspbian/ wheezy main non-free contrib

编辑此文件后，请使用 sudo apt-get update 命令，更新软件列表。
