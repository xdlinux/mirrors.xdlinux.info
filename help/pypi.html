使用说明

  1. 打开console (terminal)
  2. 在$HOME下新建文件夹.pip:mkdir ~/.pip
  3. 在~/.pip中新建配置文件pip.conf:touch ~/.pip/pip.conf
  4. 在pip.conf中添加如下内容：
    5. [global]
    index-url = http://mirrors.xdlinux.info/pypi/simple
    trusted-host=mirrors.xdlinux.info

注意

  • 上述方法仅对当前用户永久生效
  • 欲临时使用指定源进行安装(以spyder为例),可以通过命令 pip install -i http://mirrors.xdlinux.info/pypi/simple spyder 实现
